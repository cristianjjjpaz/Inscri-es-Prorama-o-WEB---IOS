Neste curso você aprenderá como criar seu próprio site, atráves das ferramentas:

  • HTML: (HyperText Markup Language)
    • Nele você apenderá a construir websites inserindo e formatando textos, imagens e vídeos, por meio de hipertextos.

  • CSS: (Cascading Style Sheet)
    • É uma linguagem usada para definir estilos em elementos unicos ou grupos de elementos na sua página web.
    Por exemplo: "Quero que o título principal, na minha página, seja mostrado como um texto grande e de cor vermelha."

  • Java Script:
    • É uma linguagem de programação que permite implementar itens complexos em páginas web mostrando conteúdo que se atualiza em um intervalo de tempo.
